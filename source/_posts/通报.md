# （一）通报的概念

&emsp;&emsp;通报是适用于表彰先进、批评错误、传达重要精神或情况的公文。

# （二）通告的特点、分类和结构要素

## 特点

1. **典型性**。无论是表彰先进、批评错误，还是传达重要精神和情况，通报内容必须具有典型性，而不是一般性的人、事、情况。
2. **教育性**。通报是为了宣传、教育广大人民群众弘扬正气、或鼓励先进、批评错误、汲取教训、警醒群众，使人真正从思想上形成正确的认识。
3. **时效性**。主要反映最近存在的新情况、新问题、新经验等，具有较强的时效性，必须抓住有利时机及时通报，才能达到宣传、教育的目的。

## 分类

&emsp;&emsp;根据内容和功用，分为以下三类。
+ **表彰性通报**。主要用于表彰和宣传先进集体、先进个人的典型事迹，总结成功经验，号召人们号召人们向先进学习。
+ **批评性通报**。主要用于批评处理重大事故、事件、违法违纪案件等，树立反面典型，告诫人们汲取教训，防止类似错误再次发生。
+ **情况性通报**。主要用于上级领导机关向所属下级机关传达有关重要情况，发布重要信息，以便下级机关及广大人民群众及时掌握有关实施方针，统一认识，协调并推动工作。

## 结构要素

&emsp;&emsp;通报一般由标题、主送机关、正文、落款构成。

### 标题

&emsp;&emsp;一般有三种形式：
+ 一是发文机关+事由+文种，如“教育部办公厅关于近期xx学校食物中毒事件的通报”；
+ 二是事由+文种，如“关于处理xx单位建房、分房中违纪事件的通报”；
+ 三是只写文种“通报”。

### 主动机关

&emsp;&emsp;指被通报者的名称。普发性的通报不写主送机关。

### 正文

&emsp;&emsp;不同类型的通报，正文结构和内容有所不同。

&emsp;&emsp;表彰性通报和批评性通报的正文结构基本一致，一般由事由、分析评价、决定事项和希望要求四部分构成。
+  事由主要概述事项的时间、地点、人物、原因等；
+ 分析评价主要是分析事项积极或消极意义及带来的正面或负面影响，表明肯定或否定态度；
+ 决定事项主要是宣布奖励和或处分的决定；
+ 希望要求主要是号召人们向被表彰的人物或事迹学习或从错误中汲取教训。

&emsp;&emsp;情况性通报的正文结构一般由三部分组成：
+ 第一部分先概述情况；
+ +第二部分主要总结经验教训，指出存在问题，说明处理结果；
+ 第三部分提出改进工作的希望和要求，有些通报可省略此部分。

### 落款

&emsp;&emsp;即发文机关名称及成文日期。

# （三）通知与通报的异同

&emsp;&emsp;通知与通报的相同之处，都属于告知性文字，要求有关单位和人员了解公文内容或配合执行，属于下行文。它们的不同之处有以下几点。
+ 第一，发文目的不同。通知在于让收文机关知道要做什么以及如何做、有哪些注意事项等；通报是让收文机关了解发生了什么事，哪些事应该提倡、表彰，哪些事应受到批评和处罚，哪些事应该提高警惕等。
+ 第二，内容不同。通知侧重提出要求，明确界限；通报侧重说明、介绍某些事物和情况，可以提出具体要求，也可不提任何要求。
+ 第三，发送对象不同。通知的对象都是与内容有直接关系的单位和个人；通报不但要发给与内容有直接关系的单位和个人，往往还发给与通报内容无直接关系的单位和个人。
